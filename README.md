# 산소 팡팡! 광합성 타이쿤

중·고등 생명과학 **'광합성에 영향을 미치는 환경 요인'** 단원을 위한 가상 실험 시뮬레이션입니다.
빛의 세기·이산화 탄소 농도·온도를 조절하며 검정말의 산소 발생량을 측정하고, 3회 평균으로 그래프를 완성합니다.

의존성 없는 **단일 HTML 파일**이라 네트워크 없이도 동작하며, 태블릿 홈 화면에 앱처럼 설치할 수 있습니다.

## 탐구 구성

| 탐구 | 조작 변인 | 통제 변인 |
|---|---|---|
| 탐구 1 | 빛의 세기 (0~100%, 10단위) | CO₂ 100%, 온도 30°C |
| 탐구 2 | CO₂ 농도 (0~100%, 10단위) | 빛 100%, 온도 30°C |
| 탐구 3 | 온도 (0~50°C, 1도 단위) | 빛 100%, CO₂ 100% |

통제 변인도 직접 조절할 수 있습니다. 다만 통제 조건이 다른 데이터는 한 그래프에 섞이지 않도록
저장 키에 조건을 포함하고, 표에서 `다른 조건` 으로 구분해 표시합니다.

## 산소 발생량 모델

```
f_L = light / (light + 15)                     # 빛 — 포화 곡선
f_C = co2   / (co2   + 15)                     # CO₂ — 포화 곡선
f_T = sin(π/2 · T/35)^1.2         (T ≤ 35)     # 효소 활성 — 35°C에서 정점
      max(0, 1 - (T-35)/10)       (T > 35)     # 효소 변성 — 45°C에서 0

발생량 = 2.5 · min(f_L, f_C) · f_T             # 최소량의 법칙
```

온도 곡선은 **적정 온도 25~35°C** 라는 생물학적 사실에 맞춰 설계했습니다
(25°C = 최대의 88%, 30°C = 97%, 35°C = 100%). 빛 또는 CO₂가 0이면 발생량은 0입니다.

측정값에는 실험 오차가 섞입니다. 기본은 **비례 오차(±5%)** 인데, 명세대로 절대 오차(±0.1)를 쓰면
저온·저광량 구간에서 오차가 신호를 압도해 그래프가 90% 확률로 뒤집히기 때문입니다.
체크박스로 절대 오차 모드로 되돌릴 수 있습니다.

## 그래프

3회 측정이 완료된 조건만 점으로 찍히고, **단조 3차 보간(Fritsch–Carlson)** 으로 곡선을 잇습니다.
일반 Catmull–Rom 스플라인은 곡선이 측정값 바깥으로 튀어 측정하지 않은 극값을 만들어내므로
쓰지 않았습니다.

## 기능

- 표(CSV)·그래프(PNG) 내보내기 — CSV는 UTF-8 BOM으로 Excel 한글 호환
- 환경에 반응하는 검정말 캐릭터 (😎 최적 / 😴 어두움 / 🥵 고온 / 😵 CO₂ 부족)
- 업적 배지, 히든 미션, 성공자 명단
- 사운드 (Web Audio 합성, 기본 꺼짐 · 음량 조절 가능)
- 라이트/다크 모드, 태블릿 대응, `prefers-reduced-motion` 지원

## 실행

`index.html` 을 브라우저로 열면 바로 실행됩니다.

홈 화면 추가(PWA)와 오프라인 캐시는 브라우저 정책상 `https` 또는 `localhost` 에서만 동작합니다.
로컬에서 시험하려면:

```bash
python -m http.server 8000    # → http://localhost:8000
```

## 개인정보

히든 미션 성공자 명단은 **각 기기의 localStorage** 에만 저장되며 서버로 전송되지 않습니다.
명단이 화면에 공개되므로 학번·전화번호처럼 보이는 입력은 차단하고, 입력창에 안내를 표시합니다.
